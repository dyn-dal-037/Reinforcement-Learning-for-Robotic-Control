# Reinforcement-Learning-for-Robotic-Control

This repository explores **reinforcement learning algorithms for robotic control**, with a focus on **Soft Actor-Critic (SAC)** and **Proximal Policy Optimization (PPO)** in continuous control environments.

The current implementation is tested on the **BipedalWalker** environment, with emphasis on understanding:
- Reward design
- Stability constraints
- Policy convergence
- Feedback-based control strategies
